About the Gaeilge Translator

This web site has been created by me, because I can never find good resources for learning Irish and specifically the translation sites did not have all the functionality I needed. In particular, I could not find a site that did all three things: Translate English -> Irish, Irish-> English, and have pronunciation for all stages of this.

There are many sites that do each of these fairly well, so I have scraped three different sites to provide the information that you see searched here.

If you're interested in web development this site was programmed using Node.js and is hosted on IBM's bluemix.


What's Coming Next

So we've got a pretty translator but what's next? You'll soon be able to save translations to your favourites, practice those saved translations, and download them to your computer.


And What About an App?

Let's face it - all the current Irish translation apps are either terrible or plain non-existent. Now for the good news - I'll be turning the functionality of this site into an app as soon as I can. Expect an Irish transltor app, with sound, available before the end of June.
